进度条工具tqdm

2019-04-29

安装

pip install tqdm

使用

tqdm包是一个python进度条工具，可以在我们进行for循环的时候显示我们的迭代进度。

例如：

from tqdm import tqdm
import time

text = ""
for char in tqdm(["a", "b", "c", "d"]):
    time.sleep(0.25)
    text = text + char

也可以用于显示文件下载进度、文件加载进度。

可以自定义进度条的加载，例如：

with tqdm(total=100) as pbar:
    for i in range(10):
        pbar.update(10)

或

pbar = tqdm(total=100)
for i in range(10):
    pbar.update(10)
pbar.close()
